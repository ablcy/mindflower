# 心流花园 - MindFlower v0.0.1

> 专注时光，心流相伴 🌸

## 📱 项目简介

「心流花园」是一款创新的时间管理 APP，将**专注心流**与**养花**完美结合。只有通过专注，你的花朵才会生长。越专注，花朵越茂盛！

### ✨ 核心特色

- 🌱 **创新玩法**：专注时间 = 花卉生长时间
- 🌻 **丰富花卉**：10种美丽花卉可选（持续更新中）
- 🌦️ **天气系统**：实时天气影响花卉生长
- 🏆 **成就系统**：解锁成就，保持专注动力
- 📊 **数据统计**：记录你的每一次专注
- 🌙 **暗色模式**：保护你的眼睛

### 1️⃣ 花卉系统

**10种花卉可选：**
- 🌻 向日葵 - 向着阳光生长
- 🌹 玫瑰 - 爱之花朵
- 🌼 菊花 - 秋日之花
- 🪷 荷花 - 出淤泥而不染
- 🌷 郁金香 - 荷兰国花
- 🌸 牡丹 - 花中之王
- 🌺 茉莉花 - 芬芳迷人
- 🌺 杜鹃花 - 映山红
- 🌾 薰衣草 - 紫色浪漫
- 🌸 兰花 - 花中君子

**6个生长阶段：**
1. 🌱 种子阶段
2. 🌿 发芽阶段
3. 🍃 长叶阶段

- 专注开始，花卉开始生长

- 天气影响生长速度

## 📱 打包成安卓 APP

### 方法一：使用 Capacitor

```bash
# 1. 安装 Node.js
# 下载：https://nodejs.org/

# 2. 安装 Capacitor
npm install -g @capacitor/core @capacitor/cli

# 3. 初始化项目
cd C:/Users/lcy/WorkBuddy/20260502143338/flower-app
npx cap init MindFlower com.mindflower.app

# 4. 添加安卓平台
npx cap add android

# 5. 构建项目
npx cap sync

# 6. 打开安卓项目
npx cap open android
```

然后在 Android Studio 中生成 APK。

### 方法二：使用 HBuilderX（最简单）

1. 下载 HBuilderX：https://www.dcloud.io/hbuilderx.html
2. 打开项目文件夹
3. 右键 → 发行 → 原生 APP-云打包
4. 等待打包完成（约 5-10 分钟）
5. 下载 APK 文件

## 💡 开发说明

### 如何修改花卉数据？

编辑 `js/flowers.js` 中的 `flowerTypes` 数组：

```javascript
{
    id: 'your_flower',
    name: '花卉名称',
    icon: '🌸',
    description: '花卉描述',
    growthStages: ['🌱', '🌿', '🍃', '🌼', '🌸', '🌸'],
    growthTime: [10, 20, 20, 20, 30], // 每个阶段需要的时间（分钟）
    bonus: { sunny: 1.2, rainy: 1.0, cloudy: 1.0 } // 天气加成
}
```

### 如何修改专注计时器默认时长？

编辑 `index.html` 中的 `default-focus-time` 输入框的 `value` 属性，或者在设置界面中修改。

### 如何接入真实天气 API？

编辑 `js/weather.js` 中的 `fetchWeather` 方法，使用真实的天气 API（如和风天气、OpenWeatherMap）。
