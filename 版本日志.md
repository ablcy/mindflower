# 版本更新日志

## v0.0.2 (2026-05-02)

### 🎨 画风全面升级 - 水墨手绘风格

#### ✨ 新增功能
- ✅ **水墨手绘花卉插图**：全部10种花卉改为中国传统水墨画风格SVG插图
- ✅ **水墨CSS主题**：全局水墨风格（宣纸底色、墨色文字、水墨印章动效）
- ✅ **滤镜效果**：SVG滤镜模拟水墨晕染、笔触质感
- ✅ **花卉SVG生成器**：`ink-wash-flowers.js` 实时生成各生长阶段的水墨花卉插图

#### 🔧 技术改进
- 移除所有 emoji 花卉图标，改用 `InkWashFlowers.createFlowerSVG()` 生成水墨SVG
- `flowers.js` 移除 `icon` 和 `growthStages` 字段，改为动态SVG渲染
- `app.js` 所有花卉渲染从 `textContent` 改为 `innerHTML` 以支持SVG
- 新增 `css/ink-wash.css` 水墨主题样式文件
- 新增 `js/ink-wash-flowers.js` 水墨花卉SVG生成器

#### 📦 仓库变更
- 仓库名改为小写：`ablcy/mindflower`（GitHub地址：https://github.com/ablcy/mindflower）

## v0.0.1 (2026-05-02)

### 🎉 初始版本

#### ✨ 新增功能
- ✅ **花卉选择系统**：10种花卉可选（向日葵、玫瑰、菊花、荷花、郁金香、牡丹、茉莉花、杜鹃花、薰衣草、兰花）
- ✅ **生长阶段系统**：6个生长阶段（种子 → 发芽 → 长叶 → 花苞 → 开花 → 茂盛）
- ✅ **专注计时器**：自由设置专注时长，开始专注后花卉开始生长
- ✅ **花园系统**：可同时养多朵花，每朵花独立计时，开花后存入花园
- ✅ **天气系统**：根据城市获取天气，天气影响花卉生长速度
  - ☀️ 晴天：生长速度 +10%~20%
  - 🌧️ 雨天：生长速度 +20%~50%
  - ☁️ 多云：生长速度 -10%~+10%
- ✅ **成就系统**：10个成就可解锁（初次开花、花园大师、专注达人等）
- ✅ **统计功能**：今日专注、本周专注、总专注时长、养花数量统计
- ✅ **设置功能**：城市设置、默认专注时长设置、白噪音播放
- ✅ **暗色模式**：支持亮色/暗色主题切换
- ✅ **2D手绘风格**：清新简洁的视觉设计

#### 🔧 技术实现
- 使用 HTML + CSS + JavaScript 开发
- 本地存储（localStorage）保存数据
- 响应式设计，支持移动端和桌面端
- SVG 计时环动画
- CSS 动画效果（生长、开花、成就解锁等）
